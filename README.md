# Insta Share App

Instagram-style social feed built with **React 19** and **Vite**.

## Tech stack

- React 19 + Vite 6
- React Router 7
- React Slick, React Icons, js-cookie

## Setup

```bash
npm install
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm start` | Same as `dev` |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
